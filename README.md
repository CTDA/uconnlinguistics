# University of Connecticut
## College of Liberal Arts and Sciences, Linguistics  

* Summary:  
This repository offers data sets for the UConn Department of Linguistics. A data sets consists of book pages' urls. The site used for the url is the [Connecticut Digital Archive Collections](http://ctdigitalarchive.org).

* Method:  
Pages were selected based on the nature of the book topic. A Solr query was used to extract the PIDs for all the pages for a book. Using these PIDs, the page url was constructed using a python script.
